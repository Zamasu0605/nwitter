routes 폴더는 페이지를 저장합니다.
어떤 화면을 보여 줄지 관리해 주는 친구 => 어떤 컴포넌트를 렌더링 할지 결정하는 역할.

jsconfig.json 파일은 리액트에 사용할 JS를 위한 설정 파일입니다.
자바스크립트 설정을 개발자가 원하는 대로 바꿀 수 있다고 합니다. => 절대 경로 설정.

안에 적은 코드의 의미는 src 폴더를 기준으로 폴더나 파일 경로를 적을 수 있도록 해놓은 것입니다. (가독성 좋아짐!)

상대 경로의 예시는 import Home from "../routes/Home"; 이고
절대 경로의 예시는 import Home from "routes/Home"; 으로, src 폴더를 기준으로 절대 경로를 지정하였다.

절대 경로는 파일 이름과 npm install로 설치한 패키지 이름과 같으면 오류를 발생시킨다.
그렇기에 firebase.js 파일을 fbase.js 파일로 이름을 바꿔주었다.
